# 0codelabs.gs
Google App Scripts
## Setup

```console
$ git clone https://github.com/freddiefujiwara/0codelabs.gs.git
$ cd 0codelabs.gs
$ npm i
```

## Login if you want to start from scratch
```console
$ npm run clasp:login # if you need
$ npm run clasp:create
```

## Create if you want to start from current gas
```console
$ npm run clasp:login # if you need
$ npm run clasp:pull
```

## Lint
Validate your code
```console
$ npm run lint
```
Fix your code
```console
$ npm run lint:fix
```

## Test

```console
$ npm run test
```
